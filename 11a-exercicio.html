Como Redirecionar para outras páginas em HTML?

Para Redirecionar em outra página, utilizamos a tag a. Com isso podemos fazer modificações para um link externo ou interno

Porém as vezes é necessario voltar para a mesma página, segue o exemplo abaixo.

Clique no link para ir para a página 11b


Voce será redirecionado para a página 11b
